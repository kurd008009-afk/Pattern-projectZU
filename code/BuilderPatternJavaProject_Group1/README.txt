Builder Pattern Project - Group 1
================================

How to run in IntelliJ IDEA:
1. Open IntelliJ IDEA.
2. Choose Open.
3. Select this project folder: BuilderPatternJavaProject_Group1
4. Wait for Maven import.
5. Open src/main/java/com/group1/builderpattern/Main.java
6. Click Run.

What the project demonstrates:
- Product: Computer
- Builder interface: ComputerBuilder
- Concrete builders: GamingPCBuilder, OfficePCBuilder, WorkstationPCBuilder
- Director: ComputerDirector
- Client: Main
